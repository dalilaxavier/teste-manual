# RF01 - Login 

## Cenário de Teste C01: Validação do fluxo de Login

### CT01: Validar login com sucesso

Descrição: Validar que, ao inserir credenciais válidas, o sistema permite a realização de login com sucesso

Pré-condição:
1. Usuário válido cadastrado no sistema.
2. Usuário está na tela de login.

Passo-a-passo:
1. Informar um e-mail válido.
2. Informar uma senha válida.
3. Clicar em Acessar.

Resultado esperado: O sistema deve redirecionar o usuário para a página home. 

Resultado obtido: O sistema redirecionou o usuário para a página home com sucesso.

Status: Passou ✅

---

### CT05: Validar login com e-mail não cadastrado

Descrição: Validação de comportamento ao tentar realizar login com o E-mail não cadastrado.

Pré-condição:
1. Usuário está na tela de login.

Passo-a-passo:
1. Informar um e-mail não cadastrado.
2. Informar uma senha válida.
3. Clicar em Acessar.

Resultado esperado: O sistema não deve autorizar o login e deve exibir uma mensagem informando a falha.

Resultado obtido: O sistema impediu o login e apresentou a mensagem "Usuário ou senha inválido. Tente novamente ou verifique suas informações!"

Status: Passou ✅
