# RF02 - Cadastro

## Cenário de Teste C02: Validação de Cadastro de usuário no sistema

### CT01: Validar cadastro com sucesso com saldo inicial

Descrição: Validar que o sistema permite a realização de cadastro com sucesso, com o saldo inicial positivo.

Pré-condição:
1. Usuário está na tela de cadastro.

Passo-a-passo:
1. Informar o nome.
2. Informar o e-mail válido.
3. Informar a senha.
4. Confirmar a senha.
5. Clicar em Criar conta com saldo.
6. Clicar em Cadastrar.

Resultado esperado: O sistema deve exibir o número da conta criada. O saldo da conta criada deve ser de R$ 1.000,00. 

Resultado obtido: O sistema exibiu o número da conta. O saldo da conta foi R$ 1.000,00.

Status: Passou ✅

2. Informar um e-mail válido.
