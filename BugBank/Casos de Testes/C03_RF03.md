# RF03 - Transferência

## Cenário de Teste C03: Validação de Transferência no sistema

### CT01: Validar transferência com o valor de transferência menor que o saldo existente

Descrição: Validar que, ao informar o valor de transferência menor que o saldo existente, o sistema permite a transferência com sucesso 

Pré-condição:
1. Usuário está na tela de transferência.
2. Saldo existente maior que o valor a transferir.

Passo-a-passo:
1. Informar uma conta válida.
2. Informar um dígito válido.
3. Informar o valor de transferência.
4. Informar a descrição.
5. Clicar em Transferir agora.

Resultado esperado: O sistema deve exibir a mensagem "Transferência realizada com sucesso", debitar o valor da conta e redirecionar o usuário para a tela Extrato. 

Resultado obtido: O sistema exibiu a mensagem "Transferência realizada com sucesso" e debitou o valor da conta, no entanto, não redirecionou para a tela de extrato.

Status: Falhou ❌

### CT11: Validar o comportamento ao transferir para a mesma conta 

Descrição: Validar que o sistema impede a realização de transferência para a mesma conta.

Pré-condição:
1. Usuário está na tela de transferência.
2. Usuário tem saldo disponível maior ou igual ao valor da transferência.

Passo-a-passo:
1. Informar o mesmo número de conta do usuário logado.
2. Informar o mesmo número de dígito do usuário logado.
3. Informar o valor de transferência.
4. Informar a descrição.
5. Clicar em Transferir agora.

Resultado esperado: O sistema deve impedir a conclusão da transferência e apresentar uma mensagem de erro, informando que não é permitido realizar transferências para a mesma conta.

Resultado obtido: O sistema impediu a conclusão da transferência e apresentou a mensagem "Nao pode transferir pra mesmo conta". 

Status: Passou ✅

Bug associado: Erro de concordância na mensagem exibida.

### CT12: Validar transferência com valor vazio

Descrição: Validar que o sistema apresenta erro ao tentar realizar transferência com o valor vazio.

Pré-condição:
1. Usuário está na tela de transferência.
2. Usuário tem saldo disponível maior ou igual do que o valor a transferir.

Passo-a-passo:
1. Informar a conta.
2. Informar o dígito.
3. Informar a descrição.
4. Clicar em Transferir agora.

Resultado esperado: O sistema deve impedir a transferência e apresentar uma mensagem de que o valor deve ser informado.

Resultado obtido: O sistema impediu a transferência e apresentou a mensagem "transferValue must be a `number` type, but the final value was: `NaN` (cast from the value `""`)". No entanto, mensagem exibida é técnica e não amigável ao usuário final. 

Status: Falhou ❌
