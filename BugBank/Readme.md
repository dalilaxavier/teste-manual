# Testes Manuais - BugBank

Este repositório contém o material utilizado para a prática de testes funcionais.

Site testado: [BugBank](https://bugbank.netlify.app/)


## Conteúdo da Pasta

- **Análise de Requisitos.md:** <br>
  Documento contendo a análise dos requisitos funcionais do sistema.

- **Plano de Testes.md:** <br>
  Documento detalhando os objetivos, critérios e técnicas de teste utilizadas.

- **Relatório de Bugs.md:** <br>
  Relatório com os bugs identificados e evidências.

- **Relatório de Testes.md:** <br>
  Relatório com resultados da execução dos casos de testes, métricas e conclusões.


## Escopo de Testes

- Login.
- Cadastro de usuário.
- Transferência.
- Extrato.


## Sobre

Este projeto foi desenvolvido como parte do meu portfólio profissional em **Quality Assurance (QA)**, com foco em:

- Testes funcionais.
- Testes positivos e negativos.
- Validação de requisitos.
- Identificação e documentação de bugs.
